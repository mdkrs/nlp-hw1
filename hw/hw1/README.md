Реализовал простой интерактивный интерфейс в темной теме, с поисковой строкой и подсказками снизу, на которые можно нажать чтобы скопировать подсказку в поле ввода.

Ключевые моменты:
- Убрал знаки препинания из подсказок, так как они сильно портили резльутат
- Добавил сохранение регистра введенного текста, чтобы если пользователь начал вводить с большой буквы, то подсказка не перетирала это
- Добавил чтобы подсказки были как и для полного последнего слова, так и для не дописанного
- По 3 вариантам дописания последнего слова начинаем искать 3 следующих слова.
